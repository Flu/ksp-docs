# KSP Documents
A repository to keep my documents, screenshots and reports about KSP missions in one-place. They are mostly LaTeX files, PDFs and screenshots.
This repository is not affiliated with ESA in any way.

## Organization
Missions and documents are organized on a per planet or per ship basis. For example, to see all the missions related to going to the Mun, you would go to `Mun/`. There are documents that don't fit a specific body, for example documents describing rockets, rovers or satellites. Those go in their own separate folders.

## Modifying
All you need is to modify the associated `.tex` file and compile it to a PDF to view it.
